# Agent Memory & Conflict Benchmarks Reference

This folder collects design references, registries, and detailed notes on benchmarks and libraries related to LLM agent memory management, stale memory retrieval, and belief conflict resolution.

## Contents

- `registry.yaml`: A curated directory of pointer links to paper resources, repositories, and datasets.
- `notes/`: Detailed architectural analyses of relevant systems, detailing their construction, evaluation protocols, and lessons for ReTrace-Bench.
