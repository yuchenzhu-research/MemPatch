# ReTrace-Bench v2 Toy Sample (sample_20_v2)

This directory contains a small, deterministic toy public sample of 4 scenarios illustrating the ReTrace-Bench v2 schema.

> [!WARNING]
> This is a **toy public sample for schema validation and baseline smoke testing only**.
> - It is **NOT** the official evaluation set.
> - It is **NOT** for training.
> - Private evaluation and gold labels for the main benchmark splits remain hidden.

## Contents
- `manifest.json`: Aggregates the split and metadata.
- `scenarios.jsonl`: Contains the event trace, memory snapshot, hidden operations, and tasks for the 4 sample scenarios.
