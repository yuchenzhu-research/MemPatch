\section{MemPatch Runtime}

The \method{} runtime is a parameter-free plug-in layer designed to convert predictions from frozen or adapted models into policy-checked, attributable, and replayable memory patches. It separates neural prediction from deterministic runtime verification.

\subsection{Plug-in Interface and Compilation}
The \method{} runtime accepts semantic proposals without editing model parameters. It normalizes heterogeneous outputs through an adapter contract $A_\Pi$:
\begin{equation}
A_\Pi: Y_\Pi \rightarrow (\widehat{s}, \widehat{Z}),
\end{equation}
where $Y_\Pi$ is the raw generation, $\widehat{s}:M\rightarrow\Sigma$ is the proposed state map, and $\widehat Z:M\rightarrow2^{I_E}$ supplies record-level provenance. The compiler constructs $\widetilde\alpha=C_V(s^0,\widehat{s},\widehat Z)$. An empty or rejected patch is a \emph{no-commit} outcome: it prevents an unauthorized write but is not claimed to repair an already stale state.

\subsection{Revision Guard}
To prevent ungrounded or contradictory actions from modifying memory, compiled action sequences are routed through the symbolic Revision Guard $G_\Gamma$:
\begin{equation}
(\alpha^+, \rho) = G_\Gamma(V, \widetilde{\alpha}),
\end{equation}
where $\alpha^+$ is the admitted canonical patch, and $\rho$ is the rejection log detailing the failure reasons. The Guard filters proposals based on a transition policy $\Gamma$ containing five predicates:
\begin{align*}
\mathcal{L}_\Gamma(V)=\{\alpha\mid
&\mathsf{Schema}(\alpha)\land\mathsf{References}_V(\alpha)\\
&\land\mathsf{Transitions}_\Gamma(\alpha)
\land\mathsf{NoConflict}(\alpha)\\
&\land\mathsf{Authorized}_\Gamma(\alpha)\}.
\end{align*}
\begin{itemize}[leftmargin=*]
\item $\mathsf{Schema}(\alpha)$: Checks that each action matches the required structure.
\item $\mathsf{References}_V(\alpha)$: Ensures all target and replacement identifiers exist in the public memory $M$ exposed by $V$.
\item $\mathsf{Transitions}_\Gamma(\alpha)$: Verifies that all state changes follow valid transitions.
\item $\mathsf{NoConflict}(\alpha)$: Rejects overlapping or contradictory actions within the same patch.
\item $\mathsf{Authorized}_\Gamma(\alpha)$: Verifies that the cited evidence identifiers match the transition rule triggers registered in $\Gamma$.
\end{itemize}
Admitted patches are projected to yield candidate state maps $s^\star = \Delta(s^0, \alpha^+)$.
Every transition outputs an audit certificate trace:
\begin{equation}
\tau = (H(V), H(\Pi), \widehat{r}, \widetilde{\alpha}, \alpha^+, \rho, s^0, s^\star),
\end{equation}
which logs the input view hash $H(V)$, predictor identity, raw proposal, candidate and admitted actions, rejection reasons, and pre/post states.

\subsection{Formal Guarantees}

We separate properties of the benchmark reference executor from properties of the runtime mediator.

\paragraph{Proposition 1 (Deterministic Reference Execution).}\label{prop:reference_determinism}
\emph{Let $E=(e_1,\ldots,e_T)$ be a finite, totally ordered event trace. If the reference step function $F_\Gamma:\Sigma_{cfg}\times E\rightarrow\Sigma_{cfg}$ is total and deterministic, then the evaluator returns a unique configuration}
\begin{equation}
C^\star=F_\Gamma(\cdots F_\Gamma(F_\Gamma(C_0,e_1),e_2)\cdots,e_T).
\end{equation}
The result follows by induction on $T$. The implementation additionally enumerates all $\binom{23}{2}=253$ unordered rule pairs in a synthetic order-swap stress test. This audit is reported as implementation coverage, not as a critical-pair confluence proof.

\paragraph{Proposition 2 (Conditional Diagnostic Identifiability).}\label{thm:diagnostic_identifiability}
\emph{Under the local linear model $\Delta=A\mathbf w+\varepsilon$, if the L3-calibrated matrix $A\in\mathbb R^{7\times7}$ has full column rank, the noiseless coefficients in the chosen corruption basis are unique: $\mathbf w=A^\dagger\Delta$. Moreover,}
\begin{equation}
\|\widehat{\mathbf{w}} - \mathbf{w}\|_2 \le \|A^\dagger\|_2 \left( \|\widehat\Delta - \Delta\|_2 + \|\widehat A - A\|_2 \|\mathbf{w}\|_2 \right).
\end{equation}
This identifies mixtures only within the seven controlled operators; it does not establish completeness of that basis for organic failures. Calibration and rank auditing use L3 only.

\paragraph{Theorem 1 (Guarded Mediation).}\label{thm:certified_transparent_mediation}
Assume (A1) $s^0\models\mathcal I_\Gamma$; (A2) the Guard admits only $\alpha\in\mathcal L_\Gamma(V)$; (A3) every legal action preserves $\mathcal I_\Gamma$; and (A4) each admitted action records a complete transition witness in $\tau$. Then:
\begin{itemize}
    \item \textbf{Mutation confinement:} only records targeted by admitted actions may change;
    \item \textbf{Invariant preservation:} $s^\star\models\mathcal I_\Gamma$;
    \item \textbf{Replayable mediation:} deterministic replay of $(s^0,\tau)$ yields $s^\star$;
    \item \textbf{Trace-complete authorization:} every committed mutation has a record-level evidence witness.
\end{itemize}
These are mediator properties conditional on A1--A4; they do not imply semantic truth or proposal completeness.

\paragraph{Corollary 1 (Deterministic Replay and Idempotency).}\label{prop:deterministic_replay}
\emph{For deterministic parsing, Guard evaluation, reconciliation, and projection, replaying the same audit trace $\tau$ satisfies $\mathsf{Replay}(s^0, \tau) = s^\star$. Replaying the same canonical patch $\alpha^\star$ is idempotent: $\Delta(\Delta(s, \alpha^\star), \alpha^\star) = \Delta(s, \alpha^\star)$.}
\textbf{Implementation Artifact:} \texttt{mempatch/tests/test\_consensus\_mode.py}

\paragraph{Corollary 2 (Local Noninterference).}\label{prop:local_noninterference}
\emph{If neither an admitted action nor an authorized reconciliation decision targets $m_i$, then $s_i^\star = s_i^0$. Malformed references and rejected actions therefore cannot silently alter unrelated records.}
\textbf{Implementation Artifact:} \path{mempatch/revision/runtime/dpa_runtime.py} (tested in \path{mempatch/tests/test_dpa_kernel.py})

\paragraph{Verifiability Metric.}
To detect logging or mediation defects empirically, the system calculates the \emph{Externally Verifiable Transition Fraction} ($\mathrm{EVTF}$):
\begin{equation}
\mathrm{EVTF} = \frac{\sum_i \mathbf{1}[s_i^\star \neq s_i^0] \mathbf{1}[\mathsf{VerifyWitness}(i, \dots)]}{\sum_i \mathbf{1}[s_i^\star \neq s_i^0]}.
\end{equation}
We report EVTF only for runs that emit the required per-transition certificate; it is not imputed for ordinary prompting baselines.
